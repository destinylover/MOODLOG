FASTAPI와 API를 이용한 간단한 기록+감정분석 앱


감정 분석 및 피드백에는 OPENAI의 CHATGPT API를 사용.

FASTAPI(PYTHON)을 사용해 백엔드구현

ORM은 SQL아케미와 DB는 SQLITE을 사용.


로그인 → 회원가입(아이디가없다면) → 일기 작성 → 감정 분석 → 감정 결과/피드백 → 일기 기록 확인 → 통계 페이지
